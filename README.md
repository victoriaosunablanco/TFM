
# Integración y consulta avanzada de datos genómicos y fenotípicos clínicos en una infraestructura Beacon FHIR interoperable en un entorno hospitalario

Este repositorio contiene el material desarrollado durante la realización del Trabajo Fin de Máster

### Contenido del repositorio

En este repositorio se incluye una carpeta que contiene los archivos relacionados con la implementación de Beacon v2 que han sido modificados durante el desarrollo del TFM.

La estructura y organización de los directorios se ha mantenido equivalente a la utilizada en el repositorio oficial de la implementación Beacon, con el objetivo de facilitar la localización de los archivos modificados y permitir una mejor trazabilidad de los cambios realizados.

Además, el repositorio incluye los siguientes archivos adicionales:

- seleccion_individuos.py
Script utilizado para la selección y filtrado de los individuos empleados en el estudio.
- Datos_necesarios_OMOP.pdf
Documento que recoge los datos y tablas del modelo OMOP necesarios para el procesamiento e integración de la información clínica.
- OMOP_mappings.pdf
Documento con los mapeos y equivalencias utilizados durante la transformación de datos entre OMOP y los formatos compatibles con Beacon v2.
Objetivo

El objetivo principal de este repositorio es recopilar los recursos, configuraciones y modificaciones realizadas durante el desarrollo del TFM, facilitando la reproducibilidad del trabajo y la documentación de la integración de datos clínicos y genómicos en Beacon v2.
